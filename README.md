# Advanced JS Homework 02

This project contains two interactive tasks built with Vite:

## 1. Timer

A timer tool with a date-time picker and notification on completion. Uses
Flatpickr for date selection and iziToast for notifications.

## 2. Snackbar

A form to create notifications with a custom delay and state
(fulfilled/rejected). Demonstrates working with Promises and dynamic UI feedback
using iziToast. Includes custom styles and SVG icons.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure

- `src/1-timer.html`, `src/js/1-timer.js`: Timer task
- `src/2-snackbar.html`, `src/js/2-snackbar.js`: Snackbar task
- `src/css/`: Styles for each task
- `src/img/`: SVG icons for notifications
